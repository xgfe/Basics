特殊标签简介

  • 标签：<command>
    涵义：指定一个用户可以调用的命令【已废弃】
    示例：Try This :Hello
  • 标签：<menu>
    涵义：代表一组用户可以操作或者激活的命令。【试验阶段，浏览器基本都不支持】
  • 标签：<progress>
    涵义：显示一个进度条
    示例：70 %
    属性：
    • value：当前值
    • max：最大值
  • 标签：<meter>
    涵义：定义已知范围或分数值内的标量测量
    示例：350 degrees
    属性：
    • value：当前值
    • max：最大值
    • min：最小值
    • low：规定被视作低的值的范围
    • high：规定被视作高的值的范围
    • optimum：规定度量的优化值
    • form：规定 <meter>元素所属的一个或多个表单
  • 标签：<spacer>
    涵义：用来在网页中插入空格【已废弃】
  • 标签：<track>
    涵义：HTML <track> 元素 被当作媒体元素—<audio> 和 <video>的子元素来使用。它允许指定计时字幕（或者基于事件的数据），例如自动处理字幕。
  • 标签：<figure>
    涵义：设置独立的流内容（图像、图表、照片、代码等等）, 经常与标题 标签<figcaption>配合使用
    示例：
  • 标签：<figcaption>
    涵义：标签定义 <figure> 元素的标题,应该被置于 <figure> 元素的第一个或最后一个子元素的位置。
    示例：同上
  • 标签：<map>
    涵义：定义一个客户端图像映射。图像映射（image-map）指带有可点击区域的一幅图像
    示例：
  • 标签：<area>
    涵义：定义图像映射中的区域,永远嵌套在 <map> 元素内部（注：图像映射指得是带有可点击区域的图像）。
    示例：同上
    属性：
    • coords：定义可点击区域（对鼠标敏感的区域）的坐标【逗号分隔的四个数字】
    • href：定义此区域的目标 URL
    • nohref：从图像映射排除某个区域【nohref】
    • shape：定义区域的形状【default/rect/circ/poly】
    • target：规定在何处打开 href 属性指定的目标 URL【_blank/_parent/_self/_top】
  • 标签：<details>
    涵义：用于描述文档或文档某个部分的细节,目前只有 Chrome 支持 <details> 标签。一般和<summary>配合使用,没有<summary>的话默认显示“详细信息”
    示例：

    All pages and graphics on this web site are the property of W3School.

  • 标签：<summary>
    涵义：包含 <details> 元素的标题，<details> 元素用于描述有关文档或文档片段的详细信息。<summary> 元素应该是 <details> 元素的第一个子元素。
    示例：
    HTML5 This document teaches you everything you have to learn about HTML 5.
