table标签属性列表

  • width:规定单元格的宽度，值为数字，单位为px。
  • align:规定表格相对周围元素对齐方式，其值为left、right、center。 可实现其他HTML元素围绕表格的效果，建议用样式代替
  • bgcolor:规定表格的背景颜色。 建议用样式代替
  • border:规定表格边框的宽度，值为数字 自带border属性样式比较丑，建议设置单元格border属性代替
  • cellpadding:规定单元格的内边距，值为数字，代表内边距的百分比 从实用角度出发，最好不要规定 cellpadding，而是使用 CSS 来添加内边距
  • cellspacing:规定单元格的外边距，值为数字，代表内边距的百分比 从实用角度出发，最好不要规定 cellspacing，而是使用 CSS 来添加内边距
  • frame:规定外部边框哪个部分是可见的。 建议用样式代替
  • rules:规定内部边框哪个部分是可见的。 建议用样式代替 frame和rules组合技：frame="box" rules="all"展示所有边框
  • summary:规定表格内容的摘要。 不会产生任何视觉变化

thead、tbody、tfoot

  • thead 标签定义表格的表头。该标签用于组合 HTML 表格的表头内容。
  • thead 元素应该与 tbody 和 tfoot 元素结合起来使用。 <thead> 内部必须拥有 <tr> 标签！
  • thead、tfoot 以及 tbody 元素使您有能力对表格中的行进行分组。当您创建某个表格时，您也许希望拥有一个标题行，一些带有数据的行，以及位于底部的一个总计行。这种划分使浏览器有能力支持独立于表格标题和页脚的表格正文滚动。当长的表格被打印时，表格的表头和页脚可被打印在包含表格数据的每张页面上。
  • 当table中省略这三个标签时，浏览器会自动在<tr>标签外层套一个<tbody>标签
  • 注意点：
    1. thead和tfoot在一张表中都只能有一个，而tbody可以有多个。
    2. tfoot必须出现在tbody前面，这样浏览器在接收主体数据之前，就能渲染表尾，有利于加快表格的显示速度。这一点对大型表格尤其重要。
    3. thead、tbody和tfoot里面都必须使用tr标签。

  • align属性：
    Month Savings
    January $100
    February $80

    align 属性可以使文本围绕在表格周围。

    在 HTML 4.01 中，不赞成使用 body 元素的 align 属性；在 XHTML 1.0 Strict DTD 中，不支持 body 元素的 align 属性。
