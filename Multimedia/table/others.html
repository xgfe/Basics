<th><td>

  • valign:规定单元格内容的垂直排列方式。取值有top、middle、bottom、baseline
  • align:规定单元格中内容的水平对齐方式。，其值为left、right、center、justify、char。
  • bgcolor:规定单元格的背景颜色。 建议用样式代替
  • axis:对单元进行分类。
  • char:规定根据哪个字符来进行内容的对齐。
  • charoff:charoff 属性设置内容相对于由 char 属性规定的字符的对齐偏移量。
  • colspan:规定规定单元格可横跨的列数。值为数字。 若colspan的值大于总列数，则该行的宽度增加到行尾
  • rowspan:规定规定单元格可横跨的行数。值为数字。 若rowspan的值大于总列数，则该行的宽度增加到列尾
  • headers:定义将表头单元与数据单元相关联的方法。
  • height:规定表格单元格的高度。 建议用样式代替
  • nowrap:规定表格单元格中的内容不换行。 建议用样式代替
  • scope:定义将表头单元与数据单元相关联的方法。
  • abbr:规定单元格中内容的缩写版本。
  • width:规定表格单元格的宽度。 建议用样式代替

<col>

  • <col>标签为表格中一个或多个列定义属性值。
  • 如需对全部列应用样式，col标签很有用，这样就不需要对各个单元和各行重复应用样式了。
  • 没有结束标签
  • 属性值有：
    1. align(水平对齐方式)
    2. 垂直对齐方式(水平对齐方式)
    3. char（根据字符对齐相关内容）
    4. charoff(规定第一个对齐字符的偏移量)
    5. span(规定col元素横跨的列数)
    6. width(规定col元素的宽度)
  • 请为 col 标签添加 class 属性。这样就可以使用 CSS 来负责对齐方式、宽度和颜色等等。

<colgroup>

  • <col>用于对表格中的列进行组合，以便对其进行格式化,其属性与<col>相同。
  • 有结束标签
  • 属性值有：
    1. align(水平对齐方式)
    2. 垂直对齐方式(水平对齐方式)
    3. char（根据字符对齐相关内容）
    4. charoff(规定第一个对齐字符的偏移量)
    5. span(规定colgroup元素横跨的列数)
    6. width(规定colgroup元素的宽度)
  • 请为 colgroup 标签添加 class 属性。这样就可以使用 CSS 来负责对齐方式、宽度和颜色等等。

<caption>

  • <col>用于定义表格标题。
  • caption 标签必须紧随 table 标签之后。您只能对每个表格定义一个标题。通常这个标题会被居中于表格之上。

示例：

  • align属性：align="right"
    Month Savings
    January $100
  • 合并单元格 rowspan="2" colspan="2"
    Month Savings Savings for holiday!
    January $50
    February $80
  • valign属性： 取值为bottom或baseline。
    Month Savings
    January $100
    February $80
  • caption标题：
    我的标题
    100 200 300
    400 500 600
