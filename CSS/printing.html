打印属性

  • page

    使用@page指定页面的打印样式，可以定制页面容器的版式，方向，边空。

    语法规则：@page <pseudo-classes> { Rules }

    • <pseudo-classes>：打印伪类:first, :left, :right
      • fisrt：打印时候的第一页
      • left：左页(装订线的左边的页面)，想象打印的是一本书，第一页为封面，是右页。
      • right：右页(装订线的右边的页面)，想象打印的是一本书，第一页为封面，是右页。
    • Rules：size, margin
      • size：<length>{1,2} | auto | [ <page-size> || [ portrait | landscape] ]。eg.size: A5 landscape;

    • demo

  • page-break-before

    用于指定打印时元素前的分页规则。

    取值
    • auto：如果需要，在对象之前插入页分割符
    • always：始终在对象之前插入页分割符
    • avoid：避免在对象前面插入页分割符

  • page-break-after

    用于指定打印时元素后的分页规则。

    取值
    • auto：如果需要，在对象之后插入页分割符
    • always：始终在对象之后插入页分割符
    • avoid：避免在对象后面插入页分割符

  • page-break-inside

    用于指定打印时元素后的分页规则。

    取值
    • auto：如果需要，在当前对象内部插入页分割符
    • avoid：避免在当前对象内部插入页分割符
