设置对象的行高。即字体最底端与字体内部顶端之间的距离- line-height

取值说明

  • normal：允许内容顶开或溢出指定的容器边界。
  • length：用长度直接指定行高
  • percentage：百分比指定行高
  • number：乘数因子指定行高

示例

  • 固定数值方式： - {line-height:20px;}

    使用固定数值的方式定义line-height，如设置行高为18px类似这样的固定数值，可能会引发文字重叠的现象。

  • 百分比方式： - {line-height:130%;}
    如果是这种情况

    使用百分比的方式定义line-height，当小于100%时会引发文字重叠的现象。

  • 因子方式： - {line-height:1.5;}
    如果是这种情况

    使用因子方式定义line-height,当小于1时会出现文字重叠现象。

注意事项

  • 该属性的所有单位的取值都不支持负值
