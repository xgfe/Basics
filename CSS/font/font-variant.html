小型大写字母属性介绍-font-variant

语法

  • font-variant：normal | small-caps
  • 默认值：normal
  • 适用于：所有元素
  • 继承性：有
  • 动画性：否
  • 计算值：指定值

说明

  • normal：指定正常字体
  • small-caps：小型的大写字母字体

示例

This is normal font.

This is small-caps font.

其他特性

1.相应的脚本特性为fontVariant,即通过el.style.fontVariant即可设置或获取字体是否为小型的大写字母

兼容性

所有浏览器均支持该属性

注意事项

  • 区别对待 text-transform
