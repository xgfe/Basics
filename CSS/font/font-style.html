字体风格属性介绍-font-style

语法

  • font-style：normal | italic | oblique
  • 默认值：normal
  • 适用于：所有元素
  • 继承性：有
  • 动画性：否
  • 计算值：指定值

说明

  • normal：指定正常字体
  • italic：指定设置斜体字，前提是该字体有设计的斜体字，否则须使用oblique。<em>，<i>实际是设置字体的font-style为italic
  • oblique：指定倾斜的文本，人为倾斜，不管该字体是否有斜体字设计

其他特性

1.相应的脚本特性为fontStyle,即通过el.style.fontStyle即可设置或获取字体风格

兼容性

所有浏览器均支持该属性
